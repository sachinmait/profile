# ⚡ Prof. (Dr.) Sachin Gupta | Faculty Feed 📡

Welcome to the central repository for my personal site—an interactive, corporate-LinkedIn-themed digital feed that showcases over **24 years of academic legacy**, cutting-edge R&D, and institutional leadership.

---

## 🛠️ Tech Stack & Architecture

This portfolio drops the traditional, dusty static-cv layout for a **high-octane, single-page dynamic application** built with clean typography and real-time state simulations.

* **UI Framework:** Tailwind CSS CDN (with bespoke theme configurations)
* **Typography Engine:** Inter (High-readability sans) + Lora (Editorial serif accentuation)
* **Icons:** FontAwesome v6.4.0 (Brand systems & action primitives)
* **State Management:** Pure Vanilla JS DOM engine (No clunky frameworks, optimized rendering pipelines)

---

## 💎 Features Built In

### 1. 📬 The Live Update Feed
A chronological post stream configured to resemble premium social updates. Includes interactive components for **Likes Counters**, **Real-Time Client-Side Comments**, and dynamic **Deep-Link Copiers** for sharing publication DOIs directly.

### 2. ⚡ Dynamic Tab Routing Engine
Zero layout shifts. Seamlessly transitions active panels between:
* `Academic Narrative` (Institutional legacy)
* `Journals` (SCIE/Scopus publication matrices)
* `Books` (Curriculum textbooks)
* `Patents` (Intellectual Property registers)
* `Mentorship` (Ph.D. Supervision matrices)

### 3. 📡 Reactive Elements
* **Search Filtration Engine:** Client-side substring filtering to instantly sort publications by keywords using standard search telemetry.
* **Skill Endorsements System:** Incremental DOM counters allowing visitors to directly upvote core engineering domains (Blockchain, WSN, Explainable AI).
* **Direct Message Terminal:** An inline mock terminal mimicking an active query queue for PhD applicants and prospective research collaborators.
